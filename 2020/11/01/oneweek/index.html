常见的Linux命令
ls 查看当前目录下的文件
ls -a 查看当前目录下的文件，包含隐藏文件
touch 文件名 当前目录下创建新的文件
mkdir 文件夹名 当前目录下创建新的文件夹
cd 文件目录 进入某个文件夹
rm -rf文件名 删除指定文件
Git提交三部曲
git add -A
git commit -m“注释”
git push origin main 第一次提交 git push 第二次以后

git push origin b 提交到其他分支
git pull 让线上仓库和本地仓库变得一致，也可以解决代码冲突问题

git remote -v 查看当前git仓库绑定的远程仓库，显示空白，则是无绑定
git remote add 仓库地址 绑定成功后，再次查看
git remote remove origin 移除绑定
Hexo提交三部曲
hexo clean
hexo generate或者hexo g
hexo deploy或者hexo d

Hexo发布新的博客
